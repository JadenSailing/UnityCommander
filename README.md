# UnityCommander
简单方便的Unity命令行工具

添加Commander脚本到场景中即可使用

Simple Unity GUI Command line tool

add command component to scene

特点：

1 使用简单 添加脚本到场景即可，波浪号(~)呼出命令行

2 交互方便 部分高亮匹配，上下箭头或Tab可自由选择，自动补全，回车可确认

3 反正我不喜欢Debug面板之类点按钮的调试方式！
